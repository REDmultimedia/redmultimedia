About The Project

I designed, coded and built the Crossfit Corona website from the ground up using WordPress. This client had an existing website that was on a platform that was really hard to customize and was a bit in responsive website standards. Turns out the previous owner of the gym had bought into this platform as it had an API integration with the scheduling system that they used at the time. However, I discovered in my research that the new owner wasn't using that scheduling system and was in fact paying way more than they should have been for their web hosting because they werenâ€™t using it.

The other thing I helped this client with was creating a vectorized version of their logo as the previous owner only had low resolution .jpgs of the original design. It also had a grunge type affect applied to it that when shrunk down made the logo look pixelated and distorted. I instead mocked up the logo without this grunge effect to make it clean and crisp and more up to date with the current logo design trends. I also create a version with the grunge for them to use on print pieces that they might want to use that variation of the logo for.

So we got them migrated to a better hosting package and built this beautiful WordPress website.
